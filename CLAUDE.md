## Project Overview

Embedded Linux course materials for Obuda University. Contains source code, drivers, device tree overlays, and lab solutions for teaching embedded Linux on Raspberry Pi.

## Directory Structure

```
embedded-linux/
├── apps/       # User-space applications
├── drivers/    # Linux kernel modules and device drivers
├── overlays/   # Device tree overlays
├── scripts/    # Build and utility scripts
└── solutions/  # Lab exercise solutions
```
